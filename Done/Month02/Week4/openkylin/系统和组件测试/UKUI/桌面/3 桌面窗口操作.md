# 桌面窗口操作

## 操作步骤

1. 打开文件管理器
2. 打开/关闭文件管理器菜单
3. 最小化/最大化/拖动/缩放
4. 关闭文件管理器

## 预期结果

能够正常打开关闭文件管理器软件

能够正常调出关闭文件管理器菜单

能够正常最小化/最大化/拖动/缩放软件

## 测试结果

和预期结果一致
